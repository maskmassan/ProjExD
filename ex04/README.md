## 逃げろこうかとん (ex04/dodge_bomb.py)
### ゲーム概要
- ex04/dodge_bomb.pyを実行すると、1600x900のスクリーンに草原が描画され、こうかとんを移動されて、飛び回る爆弾から逃げるゲーム
- こうかとんが爆弾と接触すると、ゲームオーバーになる
### 操作方法
- 矢印キーで上下左右に移動する
### 追加機能
- 爆弾の追加（黄色で速さが2固定の爆弾)
- 元の爆弾の移動の変化（不規則にしました）
- ぶつかった後にメッセージの表示
